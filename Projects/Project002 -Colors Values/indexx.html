In CSS, you can specify colors using two primary models: RGB (Red, Green, Blue), which mixes light, and HSL (Hue, Saturation, Lightness), which is more intuitive for humans. These models are expressed through various syntaxes like named colors, HEX codes, and functional notations like rgb() and hsl().

How Colors Are Made

Colors on screens are typically created by combining different amounts of primary colors or by adjusting properties like hue and brightness. The two most common models in CSS are RGB and HSL.

RGB Colors

The RGB (Red, Green, Blue) model is an additive color model. This means it creates a broad spectrum of colors by mixing different intensities of red, green, and blue light. Each color component is defined by an integer value between 0 (no intensity) and 255 (full intensity). When all three components are at their maximum (255), the result is white. When all are at their minimum (0), the result is black.

HSL Colors

The HSL (Hue, Saturation, Lightness) model is more aligned with how humans perceive color. It represents colors based on three components:

  • Hue: The type of color, represented as an angle on the color wheel (0 to 360 degrees). 0 is red, 120 is green, and 240 is blue.
  • Saturation: The intensity or purity of the color, represented as a percentage (0% is a shade of gray, 100% is the full color).
  • Lightness: The brightness of the color, also a percentage (0% is black, 50% is the normal color, and 100% is white).

Color Type Definitions

Named Colors

Definition: The simplest method, using predefined, case-insensitive color names. There are over 140 standard color names supported by modern browsers.

Details and Syntax: You just write the name of the color.

color: steelblue;

Example:

<div style="background-color: lightgray;">
  <p style="color: darkslateblue;">This is a paragraph.</p>
</div>

HEX Color

Definition: A hexadecimal representation of an RGB color. It's one of the most common ways to specify colors on the web.

Details and Syntax: The value starts with a # followed by six hexadecimal digits (0-9, a-f). The first two digits represent red, the next two green, and the last two blue (#RRGGBB). A shorthand 3-digit version (#RGB) is also possible if each pair of digits is the same (e.g., #f0c is the same as #ff00cc).

/* 6-digit syntax */
color: #4C566A;

/* 3-digit shorthand */
color: #FFF; /* same as #FFFFFF */

Example:

<div style="background-color: #ECEFF4;">
  <p style="color: #BF616A;">This is a paragraph.</p>
</div>

RGB Color

Definition: Defines a color using the rgb() function, based on the Red-Green-Blue model.

Details and Syntax: The syntax is rgb(red, green, blue). Each parameter defines the intensity of the color as an integer between 0 and 255 or as a percentage from 0% to 100%.

/* Integer values */
color: rgb(76, 86, 106);

/* Percentage values */
color: rgb(30%, 34%, 42%);

Example:

<div style="background-color: rgb(216, 222, 233);">
  <p style="color: rgb(143, 188, 187);">This is a paragraph.</p>
</div>

RGBA Color

Definition: An extension of RGB that includes an alpha channel for opacity.

Details and Syntax: The syntax is rgba(red, green, blue, alpha). The RGB values are the same as before. The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

/* 50% transparent color */
color: rgba(76, 86, 106, 0.5);

Example:

<div style="background-color: rgba(67, 76, 94, 0.5);">
  <p style="color: #ECEFF4;">This is a paragraph.</p>
</div>

HSL Color

Definition: Defines a color using the hsl() function, based on the Hue-Saturation-Lightness model.

Details and Syntax: The syntax is hsl(hue, saturation, lightness).

  • Hue: An integer from 0 to 360.
  • Saturation & Lightness: Percentages from 0% to 100%.
/* A bright, saturated blue */
color: hsl(221, 23%, 65%);

Example:

<div style="background-color: hsl(220, 13%, 95%);">
  <p style="color: hsl(348, 32%, 58%);">This is a paragraph.</p>
</div>

HSLA Color

Definition: An extension of HSL that includes an alpha channel for opacity.

Details and Syntax: The syntax is hsla(hue, saturation, lightness, alpha). The HSL values are the same as before, and the alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

/* A 50% transparent, saturated blue */
color: hsla(221, 23%, 65%, 0.5);

Example:

<div style="background-color: hsla(214, 32%, 21%, 0.7);">
  <p style="color: #D8DEE9;">This is a paragraph.</p>
</div>

HEXA (HEX + Alpha)

Definition: A hexadecimal representation of an RGBA color.

Details and Syntax: The value starts with a # followed by eight hexadecimal digits (#RRGGBBAA). The first six digits represent the color, and the last two (AA) represent the alpha channel. 00 is fully transparent and ff is fully opaque. A 4-digit shorthand (#RGBA) is also possible.

/* 8-digit syntax for a color with ~50% opacity (80 in hex) */
color: #4C566A80;

/* 4-digit syntax */
color: #05a8; /* same as #0055aa88 */

Example:

<div style="background-color: #88C0D080;">
  <p style="color: #2E3440;">This is a paragraph.</p>
</div>
