Display: Flex Property

Definition

The CSS Flexbox Layout (Flexible Box) module is a one-dimensional layout model that offers an efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown or dynamic. The main idea is to give the container the ability to alter its items' width, height, and order to best fill the available space.

Details (with Usage and properties)

Using Flexbox involves two main components: the flex container (the parent element) and the flex items (the direct children). You activate Flexbox by setting the display property of a parent element to flex or inline-flex. Once enabled, you use a set of properties on the container and the items to control the layout.

Property Explanations

Base HTML for all examples:

<div class="container">
  <div class="item item-1">...</div>
  <div class="item item-2">...</div>
  <div class="item item-3">...</div>
</div>

display

  • Definition: Defines a flex container, enabling a flex context for all its direct children.
  • Syntax: display: value;
  • Values: flex (block-level), inline-flex (inline-level).

Example: display: flex;

Effect: The child items align horizontally in a row instead of their default block behavior of stacking vertically.

flex-direction

  • Definition: Establishes the main-axis, defining the direction flex items are placed.
  • Syntax: flex-direction: value;
  • Values: row, row-reverse, column, column-reverse.

Example: flex-direction: column;

Effect: The items stack vertically from top to bottom.

justify-content

  • Definition: Defines alignment along the main axis. It distributes extra free space.
  • Syntax: justify-content: value;
  • Values: flex-start, flex-end, center, space-between, space-around, space-evenly.

Example: justify-content: space-between;

Effect: Items are evenly distributed; the first item is flush left, the last is flush right.

align-items

  • Definition: Defines alignment along the cross axis (perpendicular to the main axis).
  • Syntax: align-items: value;
  • Values: stretch, flex-start, flex-end, center, baseline.

Example: align-items: center;

Effect: Items are centered along the cross-axis (vertically in a row, horizontally in a column).

flex-grow

  • Definition: Defines the ability for a flex item to grow if necessary. It accepts a unitless value that serves as a proportion.
  • Syntax: flex-grow: number;
  • Values: A non-negative number (default 0).

Example: .item-2 { flex-grow: 1; }

Effect: Item 2 (green) will take up all available space in the container, while the other items remain at their natural size.

align-self

  • Definition: Allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.
  • Syntax: align-self: value;
  • Values: auto, flex-start, flex-end, center, baseline, stretch.

Example: .item-2 { align-self: flex-end; }

Effect: While the container aligns items to the top (flex-start), Item 2 (green) overrides this and aligns itself to the bottom of the cross-axis.
