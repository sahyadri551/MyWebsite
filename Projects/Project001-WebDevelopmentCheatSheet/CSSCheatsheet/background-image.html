Background Image Property

Definition

The background-image property sets one or more background images for an element.

Details (Usage and properties)

By default, a background image is placed at the top-left corner and repeated. Associated properties like background-repeat, background-position, and background-size give you full control over its behavior. The background shorthand property can set all of these at once.

List of CSS Properties and Possible Values

  • background-image: Sets the image using url('path/to/image.jpg').
  • background-repeat: Controls repetition (repeat, no-repeat, repeat-x, repeat-y).
  • background-position: Sets the starting position (center, top left, 50% 50%).
  • background-size: Sets the size (auto, cover, contain, 100px 50px).
  • background-attachment: Sets scrolling behavior (scroll, fixed).

Example Illustration

This div has a background image that is centered, does not repeat, and covers the entire area.

.bg-image-example {
  height: 250px;
  color: white;
  text-shadow: 2px 2px 4px black;
  background-image: url('https://via.placeholder.com/600x300');
  background-position: center;
  background-repeat: no-repeat;
  background-size: cover;
}
