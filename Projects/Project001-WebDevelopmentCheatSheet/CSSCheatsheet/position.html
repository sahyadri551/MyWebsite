Position Property

Definition

The position property can take one of five values, each altering the element's behavior within the page layout:

  • static: The default value. The element is positioned according to the normal flow of the document. The top, right, bottom, left, and z-index properties have no effect.
  • relative: The element is positioned according to the normal flow of the document, and then offset relative to itself based on the values of top, right, bottom, and left. The space it would have occupied in the normal flow is preserved.
  • absolute: The element is removed from the normal document flow, and no space is created for it in the page layout. It is positioned relative to its nearest positioned ancestor (i.e., an ancestor with a position value other than static). If no positioned ancestor exists, it is positioned relative to the initial containing block (often the <html> element).
  • fixed: The element is removed from the normal document flow and is positioned relative to the viewport. It remains in the same position even when the page is scrolled.
  • sticky: The element is treated as relative until it crosses a specified threshold, at which point it becomes fixed. It's often used for headers that should stick to the top of the screen after scrolling down.

Details (Usage and properties)

The `position` property alters an element's behavior in the page layout. It can remove an element from the normal flow, allowing it to be placed relative to its parent or the viewport.

List of CSS Properties and Possible Values

  • position:
    • static
    • relative
    • absolute
    • fixed
    • sticky
  • top, right, bottom, left: Accept length values (e.g., px, em, %) or auto.
  • z-index: Accepts an integer or auto to control the stacking order.

Example Illustration

The outer div has `position: relative`, which creates a positioning context for its children. The inner div has `position: absolute`, placing it relative to the outer div.

.parent-box {
  position: relative;
  height: 200px;
  border: 2px solid #c2185b;
}
.absolute-child {
  position: absolute;
  top: 20px;
  right: 20px;
  width: 100px;
  height: 100px;
  background-color: #fce4ec;
  border: 1px solid #e91e63;
}
