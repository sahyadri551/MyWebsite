Box Size Properties

Definition

Box sizing properties define the width and height of an element. This includes `width`, `height`, `min-width`, `max-width`, `min-height`, `max-height`, and the `box-sizing` property.

Details (Usage and properties)

These properties control an element's dimensions. `min-*` and `max-*` properties provide constraints for flexible layouts. The `box-sizing` property determines how the final dimensions are calculated, either including or excluding padding and border.

List of CSS Properties and Possible Values

  • width, height, min-width, max-width, min-height, max-height: Accept ``, ``, `auto`, `max-content`, `min-content`, or `fit-content()`.
  • box-sizing: `content-box` (default) or `border-box`.

Example Illustration

This div has a `width` of 300px but is constrained by `max-width`. With `box-sizing: border-box`, its padding and border are included in the total width.

.sizing-box {
  width: 300px;
  max-width: 80%;
  padding: 20px;
  border: 5px solid #388e3c;
  background-color: #e8f5e9;
  box-sizing: border-box;
}
This div has a width of 300px, but will not exceed 80% of its parent's width. Due to box-sizing: border-box;, the padding and border are included within the 300px width.
