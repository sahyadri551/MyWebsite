Paragraph & Layout Properties

Definition

These properties control the layout and spacing of blocks of text and other elements, managing aspects like line height, character spacing, and how elements interact with each other in the document flow.

List of CSS Properties and Possible Values

  • letter-spacing: Adjusts the space between characters.
  • word-spacing: Adjusts the space between words.
  • line-height: Sets the distance between lines of text (e.g., 1.6, 24px).
  • white-space: Controls how white space is handled (normal, nowrap, pre).
  • overflow: Manages content that overflows its container (visible, hidden, scroll, auto).
  • overflow-x / overflow-y: Controls overflow on the horizontal and vertical axes independently.
  • float: Places an element on the left or right, allowing text to wrap around it (left, right).

Example Illustration

This example combines several layout and spacing properties.

.float-left {
  float: left;
  margin-right: 15px;
}
.paragraph-example {
  overflow: auto;
  line-height: 1.6;
  letter-spacing: 0.5px;
  word-spacing: 2px;
}
This text demonstrates paragraph properties. Notice the increased line height for readability and the wider letter spacing. The image on the left is floated, so this text wraps around it. This entire container is set to 'auto' for overflow, so if the content gets too big, scrollbars will appear.
