Overflow Property

Definition

The overflow property specifies what should happen if content overflows an element's box. It controls whether to clip the content, render scrollbars, or simply let the content spill out of its container.

Details (with Usage and properties)

Overflow occurs when an element's content is too large to fit within its designated size (a specific width and height). The overflow property is a shorthand that sets the desired behavior for both the horizontal and vertical directions.

It's crucial for creating scrollable areas on a webpage and for preventing content from breaking the layout when its size is unpredictable.

List of all possible CSS properties and their possible Values

  • overflow: A shorthand property that sets both overflow-x and overflow-y.
  • overflow-x: Specifies the handling of overflow only for the horizontal (left and right) edges.
  • overflow-y: Specifies the handling of overflow only for the vertical (top and bottom) edges.

Possible Values

  • visible (Default): The content is not clipped and may be rendered outside the element's box.
  • hidden: The content is clipped, and the rest of the content is invisible. There are no scrollbars.
  • scroll: The content is clipped, and scrollbars are added to allow the user to see the rest of the content. Scrollbars will be present even if the content does not overflow.
  • auto: The content is clipped, and scrollbars are added only when necessary (i.e., when the content actually overflows). This is the most commonly used value.

Example illustration on a div element with its effect

Below is a div with a fixed size. Try changing the overflow property in your browser's developer tools to see the effect of each value.

overflow: scroll;

.overflow-demo {
    width: 400px;
    height: 150px;
    overflow: scroll; /* Change this to hidden, visible, or auto */
    /* Other styling for visualization */
    padding: 15px;
    border: 2px dashed #0d47a1;
    background-color: #bbdefb;
}
This is a long line of text inside a fixed-size container. Because the content is too long to fit, the 'overflow' property comes into play. With 'scroll', you will always see both a horizontal and vertical scrollbar, regardless of whether the content actually overflows in that direction. This text will overflow horizontally, and if you add more lines, it will overflow vertically as well, demonstrating how scrollbars let you access the hidden content.

overflow: auto;

This box uses overflow: auto. Notice that only a vertical scrollbar appears because the content only overflows in that direction. A horizontal scrollbar wouldonly appear if more lines were added. This is generally preferred over scroll.

overflow: hidden;

With overflow: hidden, any content that does not fit within the box is simply clipped and becomes invisible. There is no way for the user to access the clipped content.For example, if you add more text here, it will not be visible outside the box, and no scrollbars will appear.
