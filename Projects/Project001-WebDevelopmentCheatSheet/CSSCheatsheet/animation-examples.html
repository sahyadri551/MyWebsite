CSS Animation Showcase

A collection of 10 common and useful animation patterns created with CSS and a touch of JavaScript for interactivity.

1. Hover Effect

A simple effect that provides feedback when a user hovers over an interactive element. It uses the :hover pseudo-class and the transition property.

Hover Over Me

Notice the smooth lift and shadow effect.

/* CSS */
.hover-card {
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}
.hover-card:hover {
    transform: translateY(-10px) scale(1.03);
    box-shadow: 0 10px 25px rgba(0, 0, 0, 0.2);
}

2. Loading Spinners

Visual indicators used to show that content is being loaded. These are created with looping @keyframes animations.

/* CSS */
@keyframes spin {
    to { transform: rotate(360deg); }
}
.spinner-border {
    animation: spin 1s linear infinite;
}

@keyframes bounce {
    0%, 80%, 100% { transform: scale(0); }
    40% { transform: scale(1.0); }
}
.spinner-dots div {
    animation: bounce 1.4s infinite ease-in-out both;
}

3. Parallax Scrolling

A simple but effective parallax effect can be achieved with pure CSS by using a fixed background attachment. As you scroll, the content moves over the static background.

Parallax Effect

The background stays put!

/* CSS */
.parallax-section {
    background-image: url(...);
    background-attachment: fixed;
    background-position: center;
    background-repeat: no-repeat;
    background-size: cover;
}

4. Modal/Lightbox Entrance/Exit

Animating the appearance and disappearance of a modal window. This requires JavaScript to toggle a class, but the animation is handled by CSS transitions.

Modal Title

This is the content of the modal. It appears with a smooth fade and scale animation. Click the button or outside to close.

/* CSS */
.modal-overlay {
    opacity: 0;
    pointer-events: none;
    transition: opacity 0.3s ease;
}
.modal-overlay.is-open {
    opacity: 1;
    pointer-events: auto;
}
.modal-content {
    transform: scale(0.8);
    transition: transform 0.3s ease;
}
.modal-overlay.is-open .modal-content {
    transform: scale(1);
}

5. Hero Section Animation

Animate text elements on page load to create an engaging entrance for a key section of your site. This uses a @keyframes animation that runs once.

Animated Hero Title

This text slides in with a slight delay.

/* CSS */
@keyframes slide-up {
    to { transform: translateY(0); }
}
.hero-section h1 {
    transform: translateY(100%);
    animation: slide-up 0.8s ease forwards;
}
.hero-section p {
    transform: translateY(100%);
    animation: slide-up 0.8s 0.3s ease forwards; /* With delay */
}

6. Navigation Menu Transition

A common pattern for mobile navigation. A hamburger icon animates to an 'X' and a menu smoothly slides into view. This also uses a class toggle via JavaScript.

My Website

  • Home
  • About
  • Services
  • Contact
/* CSS */
.main-nav {
    transform: translateX(100%);
    transition: transform 0.4s cubic-bezier(0.23, 1, 0.32, 1);
}
.nav-header.nav-open .main-nav {
    transform: translateX(0);
}

7. Image Carousel/Slider

A simple, auto-playing image slider made with only CSS. It uses a @keyframes animation to change the transform property of a container element.

/* CSS */
@keyframes slide {
    0%, 20% { transform: translateX(0%); }
    25%, 45% { transform: translateX(-100%); }
    50%, 70% { transform: translateX(-200%); }
    75%, 95% { transform: translateX(-300%); }
    100% { transform: translateX(0%); }
}
.carousel-inner {
    display: flex;
    width: 400%; /* Number of images */
    animation: slide 16s infinite;
}

8. Interactive Form

Enhance user experience in forms with animated labels. This effect uses the :focus and :valid pseudo-classes to trigger a transition on the label.

/* CSS */
.form-label {
    transition: all 0.3s ease;
}
.form-input:focus + .form-label,
.form-input:valid + .form-label {
    top: -15px;
    font-size: 0.8em;
    color: #3498db;
}

9. Scroll-Triggered Animation

Make elements animate into view as the user scrolls down the page. This uses the JavaScript IntersectionObserver API to add a class when the element is visible, which then triggers a CSS transition.

Scroll Down to See Me

I was hidden until I entered the viewport.

I Animate Too!

This is a great way to draw attention to content.

/* CSS */
.scroll-animate {
    opacity: 0;
    transform: translateY(50px);
    transition: opacity 0.6s ease-out, transform 0.6s ease-out;
}
.scroll-animate.visible {
    opacity: 1;
    transform: translateY(0);
}

10. SVG Animation

Animate the strokes of an SVG to create a "drawing" effect. This is done by manipulating the stroke-dasharray and stroke-dashoffset properties with a @keyframes animation.

/* CSS */
@keyframes draw {
    to { stroke-dashoffset: 0; }
}
.circle {
    stroke-dasharray: 1000;
    stroke-dashoffset: 1000;
    animation: draw 2s 0.5s ease-in-out forwards;
}
.check {
    stroke-dasharray: 100;
    stroke-dashoffset: 100;
    animation: draw 1s 1.5s ease-in-out forwards;
}
