Outline Property

Definition

An outline is a line drawn outside an element's border. It is visually similar to a border but differs in key ways, most notably that it does not take up space in the layout.

Details (Usage and properties)

The main purpose of the outline is to highlight an element, often for accessibility (e.g., showing which element has keyboard focus). Unlike a border, an outline does not affect the element's dimensions or push other elements away.

List of CSS Properties and Possible Values

  • outline: Shorthand for outline-width, outline-style, and outline-color.
  • outline-width: Sets the thickness of the outline (e.g., 2px, thin).
  • outline-style: Sets the line style (e.g., solid, dotted, dashed, auto).
  • outline-color: Sets the color of the outline.
  • outline-offset: Sets the space between the outline and the border. Can be positive or negative.

Example Illustration

This div has a blue outline that is offset from the border.

.outline-example {
  padding: 20px;
  margin: 20px;
  border: 2px solid #1565c0;
  background-color: #bbdefb;
  outline: 4px dashed #0d47a1;
  outline-offset: 8px;
}
This box has a border, and an outline is drawn 8px outside of that border. Notice the outline doesn't push other elements away.
