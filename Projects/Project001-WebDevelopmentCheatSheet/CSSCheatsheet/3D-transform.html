3D Transform Property

Definition

3D transformations modify an element's position and shape in a three-dimensional space, defined by the X (horizontal), Y (vertical), and Z (depth) axes. The transform property, combined with a perspective on a parent element, allows for creating depth and rotating elements in 3D space. These transformations do not affect the document's layout flow.

List all transformation types

The following are the core 3D transform functions. Each example will apply the transform on hover with a 200ms delay as styled in the CSS. To properly view the 3D effect, a perspective property is applied to the container of each card.

translateX()

  • Definition: Moves an element horizontally along the X-axis.
  • Syntax: transform: translateX(tx);
  • Property Values: Accepts a length (px, em) or percentage (%). A positive value moves the element to the right, and a negative value moves it to the left.
  • Example: Hover over the card to move it 50 pixels to the right.
.container:hover .card { transform: translateX(50px); }

Translate X (Horizontal)

translateY()

  • Definition: Moves an element vertically along the Y-axis.
  • Syntax: transform: translateY(ty);
  • Property Values: Accepts a length (px, em) or percentage (%). A positive value moves the element down, while a negative value moves it up.
  • Example: Hover over the card to move it up by 50 pixels.
.container:hover .card { transform: translateY(-50px); }

Translate Y (Vertical)

translateZ()

  • Definition: Moves an element along the Z-axis, creating the illusion of depth. A parent element must have the perspective property for this effect to be visible.
  • Syntax: transform: translateZ(tz);
  • Property Values: Accepts a length unit (e.g., px). A positive value moves the element "closer" to the viewer, while a negative value moves it "further" away.
  • Example: Hover over the card to bring it 100 pixels closer.
.container:hover .card { transform: translateZ(100px); }

Translate Z (Depth)

rotateX()

  • Definition: Rotates an element around the horizontal X-axis. A perspective property on a parent element is required to see the 3D effect.
  • Syntax: transform: rotateX(angle);
  • Property Values: Accepts an angle unit like degrees (deg) or radians (rad). A positive value rotates the top edge towards the viewer, while a negative value rotates it away.
  • Example: Hover to rotate the card 60 degrees around the X-axis.
.container:hover .card { transform: rotateX(60deg); }

Rotate X

rotateY()

  • Definition: Rotates an element around the vertical Y-axis. A perspective property on a parent element is required to see the 3D effect.
  • Syntax: transform: rotateY(angle);
  • Property Values: Accepts an angle unit (e.g., deg). A positive value rotates the right edge towards the viewer, while a negative value rotates it away.
  • Example: Hover to rotate the card 60 degrees around the Y-axis.
.container:hover .card { transform: rotateY(60deg); }

Rotate Y

rotateZ()

  • Definition: Rotates an element around the Z-axis. This is equivalent to the 2D rotate() function.
  • Syntax: transform: rotateZ(angle);
  • Property Values: Accepts an angle unit (e.g., deg). Positive values cause a clockwise rotation, and negative values cause a counter-clockwise rotation.
  • Example: Hover to rotate the card 45 degrees.
.container:hover .card { transform: rotateZ(45deg); }

Rotate Z

scaleX()

  • Definition: Resizes an element's width along the X-axis.
  • Syntax: transform: scaleX(sx);
  • Property Values: A unitless number where 1 is the original width. Values greater than 1 stretch the width, while values between 0 and 1 shrink it.
  • Example: Hover to stretch the card's width by 30%.
.container:hover .card { transform: scaleX(1.3); }

Scale X

scaleY()

  • Definition: Resizes an element's height along the Y-axis.
  • Syntax: transform: scaleY(sy);
  • Property Values: A unitless number where 1 is the original height. Values greater than 1 stretch the height, while values between 0 and 1 shrink it.
  • Example: Hover to shrink the card's height by 20%.
.container:hover .card { transform: scaleY(0.8); }

Scale Y

scaleZ()

  • Definition: Resizes an element along the Z-axis. This does not change the visual size of a 2D element unless used in combination with other 3D transforms. It can affect how child elements are transformed in a 3D context.
  • Syntax: transform: scaleZ(sz);
  • Property Values: A unitless number where 1 is the original depth.
  • Example: Hover to scale the card's depth. This is not visually apparent on a flat object, but the text is moved forward and scaled to demonstrate the effect.
.card:hover { transform: rotateY(45deg) scaleZ(2.5); }

Scale Z (with Rotate)

matrix3d()

  • Definition: Defines a 3D transformation using a 4x4 matrix of 16 values. This single function can combine all other 3D transforms (translate, scale, rotate, and skew) into one declaration.
  • Syntax: transform: matrix3d(n,n,n,n, n,n,n,n, n,n,n,n, n,n,n,n);
  • Property Values: Sixteen unitless numbers that represent the 3D transformation matrix. Modifying these values requires a strong understanding of linear algebra.
  • Example: Hover to apply a complex transform combining rotation and translation using matrix3d().
.container:hover .card { transform: matrix3d(0.866, 0.5, 0, 0, -0.5, 0.866, 0, 0, 0, 0, 1, 0, 50, 20, 0, 1.2); }
