Box Shadow Property

Definition

The box-shadow property adds shadow effects around an element's frame, which can be used to give an element a sense of depth or to highlight it.

Details (Usage and properties)

A shadow is defined by X and Y offsets, a blur radius, a spread radius, and a color. You can also specify an inset keyword to make the shadow an inner shadow. Multiple shadows can be applied to one element by separating them with commas.

List of CSS Values

  • inset (optional): Changes the shadow to an inner shadow.
  • offset-x: Horizontal offset.
  • offset-y: Vertical offset.
  • blur-radius (optional): The larger the value, the more blurred the shadow.
  • spread-radius (optional): Positive values expand the shadow; negative values shrink it.
  • color: The color of the shadow.

Example Illustration

This div has a soft, gray drop shadow.

.shadow-example {
  padding: 25px;
  margin: 30px;
  background-color: white;
  border-radius: 8px;
  /* offset-x | offset-y | blur-radius | color */
  box-shadow: 10px 10px 15px rgba(0, 0, 0, 0.2);
}
This box has a soft drop shadow that is offset by 10px horizontally and vertically, with a 15px blur.
