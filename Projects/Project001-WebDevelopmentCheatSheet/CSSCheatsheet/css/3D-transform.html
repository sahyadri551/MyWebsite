3D Transform Properties

Definition

3D transformations add a Z-axis to the 2D plane, allowing you to manipulate elements in three-dimensional space. This creates a sense of depth and perspective, enabling effects like flipping, rotating in space, and moving elements closer to or further from the viewer.

For 3D transforms to have a visible effect, the containing element often needs a perspective property.

List all transformation types

The following are the core 3D transform functions. Each example will apply the transform on hover with a 200ms delay.

rotateX()

  • Definition: Rotates an element around the horizontal X-axis, creating a "flip" effect from top to bottom.
  • Syntax: transform: rotateX(angle);
  • Property Values: An angle unit (e.g., deg).
  • Example: Hover to rotate the card 45 degrees around its X-axis.
.container:hover .card { transform: rotateX(45deg); }

Rotate X

rotateY()

  • Definition: Rotates an element around the vertical Y-axis, like a revolving door.
  • Syntax: transform: rotateY(angle);
  • Property Values: An angle unit (e.g., deg).
  • Example: Hover to rotate the card 45 degrees around its Y-axis.
.container:hover .card { transform: rotateY(45deg); }

Rotate Y

translateZ()

  • Definition: Moves an element along the Z-axis, making it appear closer or further away. A positive value brings it "out of the screen" towards the viewer, making it look larger.
  • Syntax: transform: translateZ(tz);
  • Property Values: A length unit (e.g., px). Requires a perspective property on a parent element to be visible.
  • Example: Hover to move the card 100px closer to you.
.container:hover .card { transform: translateZ(100px); }

Translate Z

scaleZ()

  • Definition: Scales an element along the Z-axis. This does not change the element's 2D size but can affect how it transforms in 3D space when combined with other 3D functions like rotations.
  • Syntax: transform: scaleZ(s);
  • Property Values: A unitless number. This effect is often subtle and is best seen in combination with other transforms.
  • Example: Hover to apply scaleZ(2) combined with rotateY(60deg) to see the exaggerated perspective.
.container:hover .card { transform: rotateY(60deg) scaleZ(2); }
