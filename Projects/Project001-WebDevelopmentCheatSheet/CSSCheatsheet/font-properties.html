Font Properties

Definition

Font properties are used to define the typeface, size, weight, and style of the text within an element. They control the fundamental appearance of the font itself.

List of CSS Properties and Possible Values

  • font-family: Specifies a prioritized list of font names (e.g., "Arial", sans-serif).
  • font-size: Sets the size of the font (e.g., 16px, 1.2em).
  • font-weight: Sets the thickness of the font (normal, bold, 100-900).
  • font-style: Sets the style of the font (normal, italic, oblique).
  • font-variant: Allows for small-caps (normal, small-caps).
  • font-stretch: Makes text narrower or wider (condensed, expanded, normal).
  • font: A shorthand for setting most of the above properties in one line.

Example Illustration

This example changes the font family, size, weight, and style.

.font-example {
  font-family: Georgia, serif;
  font-size: 18px;
  font-weight: bold;
  font-style: italic;
}
This text uses the "Georgia" font. It is slightly larger, bold, and italicized to demonstrate font properties.
