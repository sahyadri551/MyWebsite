2D Transform Properties

Definition

2D transformations alter elements on a two-dimensional plane defined by the X (horizontal) and Y (vertical) axes. These functions are passed as values to the transform property to change an element's position, rotation, and shape without disrupting the normal document flow.

List all transformation types

The following are the core 2D transform functions. Each example will apply the transform on hover with a 200ms delay.

translateY()

  • Definition: Moves an element vertically along the Y-axis.
  • Syntax: transform: translateY(ty);
  • Property Values: Accepts a length (px, em) or percentage (%). Positive values move it down, negative values move it up.
  • Example: Hover over the card to move it down by 50 pixels.
.container:hover .card { transform: translateY(50px); }

Translate Y (Vertical)

translateX()

  • Definition: Moves an element horizontally along the X-axis.
  • Syntax: transform: translateX(tx);
  • Property Values: Accepts a length (px, em) or percentage (%). Positive values move it right, negative values move it left.
  • Example: Hover over the card to move it 50 pixels to the right.
.container:hover .card { transform: translateX(50px); }

Translate X (Horizontal)

rotate()

  • Definition: Rotates an element around its origin point (by default, the center).
  • Syntax: transform: rotate(angle);
  • Property Values: Accepts an angle unit like degrees (deg), gradians (grad), radians (rad), or turns (turn). Positive values rotate clockwise, negative values rotate counter-clockwise.
  • Example: Hover to rotate the card 15 degrees clockwise.
.container:hover .card { transform: rotate(15deg); }

Rotate

scale()

  • Definition: Resizes an element. It can take one value to scale uniformly or two values to scale the width and height independently.
  • Syntax: transform: scale(sx, sy); or transform: scale(s);
  • Property Values: A unitless number. 1 is the original size. Values greater than 1 enlarge the element, while values between 0 and 1 shrink it.
  • Example: Hover to make the card 20% larger.
.container:hover .card { transform: scale(1.2); }

Scale

skew()

  • Definition: Skews or distorts an element along the X and/or Y axes.
  • Syntax: transform: skew(ax, ay);
  • Property Values: An angle (e.g., deg). The first value skews along the X-axis, the second along the Y-axis. If only one value is given, it only skews the X-axis.
  • Example: Hover to skew the card by 10 degrees horizontally and 5 degrees vertically.
.container:hover .card { transform: skew(10deg, 5deg); }
