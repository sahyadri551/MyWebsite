Text Properties

Definition

Text properties are used to style the text content within an element. They control appearance aspects like color, alignment, decoration, and spacing, but not the font face itself.

List of CSS Properties and Possible Values

  • color: Sets the color of the text.
  • text-align: Specifies the horizontal alignment of text (start,end,auto,left, right, center, justify).
  • text-decoration: Adds decoration lines to text (underline, overline, line-through).
  • text-decoration-line: Specifies the type of line (underline, overline, line-through,).
  • text-decoration-color: Sets the color of the decoration line.
  • text-decoration-style: Sets the style of the line (solid, wavy, dotted, dashed, double).
  • text-decoration-thickness: Sets the thickness of the line.
  • text-transform: Controls the capitalization of text (uppercase, lowercase, capitalize).
  • text-shadow: Adds a shadow to the text(similar to box-shadow).
  • text-indent: Indents the first line of a text block.
  • text-align-last: Sets the alignment of the last line of a justified text block.
  • text-justify: Specifies the justification method when text-align is justify(auto,inter-word,inter-character,distribute,newspaper).
  • text-orientation: Defines the orientation of text in a line (for vertical writing modes).Possible values are:mixed,upright,sideways,sideways-right.
  • text-overflow: Specifies how to signal truncated text (clip, ellipsis).

Example Illustration

This example demonstrates multiple text properties applied to a block of text.

.text-example {
  color: #4a148c;
  text-align: justify;
  text-transform: uppercase;
  text-decoration-line: underline;
  text-decoration-color: #e91e63;
  text-decoration-style: wavy;
  text-shadow: 1px 1px 2px rgba(0,0,0,0.2);
}
This is an example of text styling. This line is justified. The text is uppercase with a wavy pink underline and a subtle shadow.
