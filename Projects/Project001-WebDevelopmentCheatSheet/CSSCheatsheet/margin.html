Margin Property

Definition

Margin is the transparent space outside an element's border. It separates the element from other elements on the page.

Details (Usage and properties)

Margins are used to control the spacing between elements. Unlike padding, an element's background does not extend into the margin. A key feature is "margin collapsing," where the vertical margins of adjacent block-level elements combine into a single, larger margin.

List of CSS Properties and Possible Values

  • margin: Shorthand for all four sides. Accepts 1 to 4 values.
  • margin-top: Sets the top margin.
  • margin-right: Sets the right margin.
  • margin-bottom: Sets the bottom margin.
  • margin-left: Sets the left margin.
  • Possible Values: <length> (e.g., 20px), <percentage>, auto (useful for centering).

Example Illustration

This div has a 30px margin, creating a visible gap around it.

.margin-box {
  width: 300px;
  padding: 20px;
  border: 2px solid #2e7d32;
  background-color: #c8e6c9;
  margin: 30px;
}
This box has a 30px margin on all sides, creating a gap between it and any neighboring elements.

Note: Margins can collapse, meaning if two vertical margins meet, they will combine into the larger of the two instead of adding together.

Note: Margins are transparent, allowing the background of parent elements to show through.
