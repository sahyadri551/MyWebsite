Transform Property

Definition

The transform CSS property lets you modify the coordinate space of the CSS visual formatting model. In simpler terms, it allows you to rotate, scale, skew, or translate (move) an element, without affecting the position of surrounding elements.

The transition-delay property specifies a delay (in seconds or milliseconds) for the start of a transition effect.

Details (with Usage and properties)

Transforms are powerful for creating visual effects and animations. Unlike changing layout properties like margin or top, transforms are typically handled by the computer's graphics processor (GPU), which results in smoother animations that are less likely to slow down the page.

When you apply a transform, the element keeps its original space in the document flow. Other elements are not pushed or moved to accommodate the transformed element's new appearance.

Give description on 2D and 3D transformation

2D Transformations

These transformations operate on a two-dimensional plane (X and Y axes). They allow you to move an element left/right and up/down, rotate it clockwise or counter-clockwise, and scale or skew it within that 2D plane.

3D Transformations

These transformations add a third dimension, the Z-axis, which represents depth (closer to or further from the viewer). This allows for effects like flipping elements over, rotating them in 3D space, and creating a sense of perspective.

List all transformation types and short note on them

  • translate(): Moves an element along the X and Y axes from its current position.
  • rotate(): Rotates an element clockwise or counter-clockwise around a given point.
  • scale(): Changes the size of an element, making it larger or smaller.
  • skew(): Skews or distorts an element along the X and Y axes.
  • matrix(): A powerful function that combines all 2D transforms into one mathematical expression.

List of all possible CSS properties and their possible Values

  • transform: Applies a 2D or 3D transformation to an element. Its values are one or more transform functions, like rotate(45deg) or scale(1.5).
  • transition-delay: Sets a time to wait before a transition effect starts. Its value is a time unit, such as 0.5s (seconds) or 200ms (milliseconds).
  • transform-origin: Changes the position of the origin point around which a transformation is applied. The default is 50% 50% (the center of the element). Values can be keywords (left, top, center) or length/percentage values.

Example illustration on a div element containing a paragraph and image with its effect

Hover over the card below. After a 200ms delay (set by transition-delay), it will rotate and scale up slightly. Notice how it does not push the text around it.

/* Add this style on :hover */
.transform-card-container:hover .transform-card {
    transform: scale(1.1) rotate(5deg);
}

.transform-card {
    transition-property: transform;
    transition-delay: 0.2s;
}
