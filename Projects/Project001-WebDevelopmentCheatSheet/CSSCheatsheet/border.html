Border Property

Definition

A border is a line drawn around the padding of an element. It visually separates the element from its surroundings.

Details (Usage and properties)

Borders are defined by their width, style, and color. The border shorthand property is often used to set all three values for all four sides at once. You can also target individual sides (e.g., border-top) or properties (e.g., border-color).

List of CSS Properties and Possible Values

  • border: Shorthand for border-width, border-style, and border-color.
  • border-width: Sets the thickness of the border (e.g., 5px, thin, medium).
  • border-style: Sets the line style. Possible values: solid, dotted, dashed, double, groove, ridge, inset, outset, none.
  • border-color: Sets the color of the border (e.g., red, #ff0000).
  • border-radius: Used to create rounded corners.

Note: Border can also be defined using individual properties for each side (e.g., border-top, border-right, border-bottom, border-left).

Example Illustration

This div has a thick, dashed, red border.

.border-box {
  width: 300px;
  padding: 20px;
  border: 5px dashed #c62828;
  background-color: #ffcdd2;
}
This box is enclosed by a 5px, dashed, red border. The border sits outside the padding but inside the margin.
