Media Query

Definition

A Media Query is a CSS technique that uses the @media rule to include a block of CSS properties only if a certain condition is true. These conditions typically relate to the characteristics of the device rendering the content, such as its screen width, height, or orientation. Media queries are the cornerstone of Responsive Web Design (RWD).

Details (with Usage and properties)

Media queries allow you to tailor your site's presentation to specific "breakpoints" or device features. The syntax consists of an optional media type (screen, print, all) and any number of media feature expressions, which test for specific conditions.

@media media-type and (media-feature) {
  /* CSS rules to apply when the condition is met */
  .selector {
    property: value;
  }
}

List of all possible CSS features

FeatureDescription
widthThe width of the viewport. Can be prefixed with min- or max-.
heightThe height of the viewport. Can be prefixed with min- or max-.
aspect-ratioThe ratio of the viewport's width to its height.
orientationThe orientation of the viewport (portrait or landscape).
prefers-color-schemeDetects if the user has requested a light (light) or dark (dark) color theme.
prefers-reduced-motionDetects if the user has requested the system minimize non-essential motion.
hoverChecks if the primary input mechanism allows the user to hover over elements.
pointerChecks the presence and accuracy of a pointing device (none, coarse, fine).
resolutionThe pixel density of the output device (e.g., in dpi).

List All logical operators

OperatorDescription
andCombines multiple media features. All conditions must be true.
, (comma)Acts as an OR operator. Applies styles if any of the queries are true.
notNegates a media query, reversing its meaning.
onlyUsed to hide styles from older browsers. Has no effect on modern browsers.

Example Illustrations

Example 1: Responsive Layout with min-width

This example demonstrates the most common use of media queries: changing a layout based on screen width. Resize your browser window to see the effect.

Effect:
- On screens narrower than 768px, the box is blue and its content is stacked vertically.
- On screens 768px or wider, the box turns green and the content is arranged side-by-side.
