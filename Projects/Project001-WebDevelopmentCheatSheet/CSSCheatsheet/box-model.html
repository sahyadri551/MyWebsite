CSS Box Model

Definition and Details

In CSS, every HTML element is a rectangular box. The CSS Box Model defines how these boxes are composed, describing the layout and space an element occupies. It consists of layers: content, padding, border, and margin.

Box Model Components:

  • Content: The actual content of the box, where text and images appear. Its dimensions are defined by width and height properties.
  • Padding: The transparent area around the content, inside the border. It creates space between the content and the border.
  • Border: A line that goes around the padding and content. It has properties for style, width, and color.
  • Margin: The transparent area outside the border. It creates space between this box and other elements.
  • Box Sizing: The box-sizing property determines how the width and height of an element are calculated, either including or excluding padding and border.

Content (Width and Height)

The content area holds the element's primary content. Its size is controlled by width and height properties.

.content-box {
  width: 300px;
  height: 150px;
  background-color: #e0f7fa;
  border: 2px solid #00796b;
}
This box has a content width of 300px and a height of 150px.

Box Sizing

The box-sizing property changes how total width/height are calculated. border-box is the modern standard, making layouts more intuitive by including padding and border in the element's defined width and height.

.box {
  width: 300px;
  padding: 20px;
  border: 5px solid #6a1b9a;
}
.content-box-sizing {
  box-sizing: content-box; /* Total width: 300 + 40 + 10 = 350px */
}
.border-box-sizing {
  box-sizing: border-box; /* Total width: 300px */
}
I use content-box. My total width is 350px.
I use border-box. My total width is 300px.
