Padding Property

Definition

Padding is the space between an element's content and its border. It is transparent and is affected by the element's background-color.

Details (Usage and properties)

Padding is used to create "breathing room" for the content inside an element. You can set padding for all four sides at once using the padding shorthand, or individually with padding-top, padding-right, etc.

List of CSS Properties and Possible Values

  • padding: Shorthand for all four sides. Accepts 1 to 4 values.
  • padding-top: Sets the top padding.
  • padding-right: Sets the right padding.
  • padding-bottom: Sets the bottom padding.
  • padding-left: Sets the left padding.
  • Possible Values: <length> (e.g., 20px, 1em), <percentage>.

Example Illustration

This div has 25 pixels of padding on all sides, pushing the text away from the border.

.padding-box {
  width: 300px;
  padding: 25px;
  background-color: #fff9c4;
  border: 2px solid #fbc02d;
}
This content is pushed away from the border by 25 pixels of padding on all sides. Notice the background color extends into the padding area.
