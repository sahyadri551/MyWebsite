CSS Units

Definition

CSS units specify the size of properties for elements, such as `width`, `height`, `margin`, `padding`, and `font-size`. They are categorized into absolute and relative units.

Details (Usage and properties)

Absolute units are fixed and do not change based on the parent element or viewport size. They are useful for print layouts or when exact dimensions are required.

Relative units are flexible and scale in relation to another value, like the parent's font size or the viewport size. They are essential for creating responsive designs.

List of CSS Properties and Possible Values

  • Absolute Units:
    • px: Pixels (1px = 1/96th of 1in)
    • pt: Points (1pt = 1/72th of 1in)
    • pc: Picas (1pc = 12pt)
    • in: Inches
    • cm: Centimeters
    • mm: Millimeters
  • Relative Units:
    • %: Percentage of the parent element's size
    • em: Relative to the font size of the element (1em = current font size)
    • rem: Relative to the font size of the root element (1rem = root font size)
    • vw: Viewport width (1vw = 1% of the viewport width)
    • vh: Viewport height (1vh = 1% of the viewport height)
    • vmin: Minimum of viewport width or height (1vmin = 1% of the smaller dimension)
    • vmax: Maximum of viewport width or height (1vmax = 1% of the larger dimension)
    • ch: Width of the "0" (zero) character in the element's font
    • ex: Height of the letter "x" in the element's font

Example Illustration

This div demonstrates various units. Its width is relative to the viewport, its font size is relative to the root, and its padding is relative to its own font size.

.units-box {
  width: 50vw; /* 50% of the viewport width */
  font-size: 1.2rem; /* 1.2 times the root font size */
  padding: 2em; /* 2 times its own font size */
  background-color: #e0f7fa;
  border: 2px solid #00796b;
}
This div's width is 50% of the viewport width. Its font size is 1.2 times the root font size, and its padding is twice its own font size.
