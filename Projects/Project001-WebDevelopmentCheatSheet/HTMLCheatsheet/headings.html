HTML Headings

Definition

HTML headings are titles or subtitles used to structure a webpage. They are defined with the <h1> to <h6> tags, where <h1> is the most important and <h6> is the least.Heading tags primarily use Global Attributes. They do not have unique attributes specific to them.

List of Heading Tags

  • <h1>: The main title of the page.
  • <h2>: A main section heading.
  • <h3>: A sub-section heading.
  • <h4>: A sub-sub-section heading.
  • <h5>: A lower-level heading.
  • <h6>: The lowest-level heading.

Example Illustration

The code below demonstrates a proper heading hierarchy.

<h1>Main Title of the Page</h1>
<p>This is the introduction to the main topic.</p>
<h2>First Main Section</h2>
<p>Content for the first main section goes here.</p>
<h3>A Sub-section</h3>
<p>Content for the sub-section goes here.</p>
