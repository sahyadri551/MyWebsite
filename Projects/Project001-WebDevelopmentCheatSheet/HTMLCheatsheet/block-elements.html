Block Elements

Definition

A block-level element always starts on a new line and takes up the full width available to it. It creates a "block" on the page and is used for structuring the main parts of your content.

Characteristics of Block Elements:

  • Start on a new line.
  • Take up the full width of their parent container.
  • Can contain other block elements and inline elements.
  • Commonly used for layout and structure.
  • All block-level elements have a "box model," meaning you can set their width, height, margin, padding, and border properties.

List of Common Block Elements

  • <div>, <pre>: A generic container for flow content.
  • <p>: A paragraph of text.
  • <h1> - <h6>: Heading levels.
  • <ul>, <ol>, <li>: List elements.
  • <header>, <footer>, <section>, <article>: Semantic layout tags.
  • <form>: A container for form controls.

Example Illustration

The following code shows two block elements. Notice how each one occupies its own line.

<div style="border: 1px solid blue;">
    This div takes up the full width.
</div>
<p style="border: 1px solid red;">
    This paragraph starts on a new line.
</p>
