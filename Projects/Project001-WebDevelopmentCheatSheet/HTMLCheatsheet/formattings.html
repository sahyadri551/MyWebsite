HTML Formatting

Definition

HTML contains several elements for defining text with a special meaning or structure. These tags are used to format the appearance of text, either for presentational purposes or to convey semantic importance.

Details & Usage

Formatting tags can be broadly divided into two categories: Physical tags (like <b> and <i>) define the visual appearance of text (e.g., bold, italic). Semantic (or logical) tags (like <strong> and <em>) provide extra meaning to the text, which is important for accessibility and SEO. For instance, <strong> indicates that the text is important, while <b> just makes it bold.

List of Formatting Tags & Examples

<b> and <strong>
<b> makes text bold without extra importance. <strong> indicates that text is important. Browsers render both as bold.
<i> and <em>
<i> makes text italic for an alternate voice (e.g., a technical term). <em> is used to emphasize text. Browsers render both as italic.
<u> and <ins>
<u> renders as underlined text, but should be avoided to prevent confusion with links. <ins> defines inserted text and is semantically more appropriate for marking additions.
<s> and <del>
<s> renders text with a strikethrough to indicate it's no longer correct. <del> defines deleted text and is semantically better for marking removals.
<mark>
The <mark> tag is used to highlight text, like using a marker pen.
<sub> and <sup>
Used for subscript (like in H₂O) and superscript (like in E=mc²).
<small>
Makes text smaller, often used for fine print or copyright notices.
<code>, <kbd>, <samp>, <var>
These are for computer-related text. <code> for code snippets, <kbd> for keyboard input, <samp> for sample output, and <var> for variables. They usually render in a monospace font.
<blockquote> and <q>
<blockquote> is for long quotations that are indented. <q> is for short, inline quotations that are enclosed in quotes.
<abbr> and <dfn>
<abbr> defines an abbreviation like WHO. <dfn> defines the defining instance of a term.
<address>
Defines contact information (author, owner) for a document or article.
