HTML Non-Closing Elements

Definition

Non-closing elements, also known as void elements or empty elements, are HTML tags that do not have a closing tag. Since they can't contain any content (like text or other elements), they don't need a closing tag. They are written with a single tag, for example <br>.

Details & Usage

Void elements are used to insert or embed something into the document at a specific point. For example, <img> embeds an image, <hr> inserts a thematic break (a horizontal line), and <input> inserts a form control. Their behavior is defined entirely by their attributes.

List of Common Non-Closing Tags & Attributes

In addition to Global Attributes, these elements have important specific attributes:

  • <br>: Inserts a single line break. It has no attributes.
  • <hr>: Represents a thematic break or horizontal rule between paragraphs. It has no specific attributes.
  • <img>: Embeds an image.
    • src: (Required) The path to the image file.
    • alt: (Required) Alternative text for the image, crucial for accessibility.
    • width / height: The dimensions of the image.
  • <link>: Links the HTML document to an external resource, most commonly a CSS stylesheet.
    • href: The URL of the linked resource.
    • rel: Specifies the relationship (e.g., 'stylesheet', 'icon').
    • type: The MIME type of the resource (e.g., 'text/css').
  • <meta>: Provides metadata about the HTML document.
    • charset: Specifies the character encoding (e.g., 'UTF-8').
    • name / content: Used for descriptions, keywords, and viewport settings.
  • <input>: (Also a form element) Creates interactive controls for web-based forms.
    • type: Defines the input type (e.g., text, radio, submit).
    • name, value, placeholder, etc.
  • <source>: Specifies multiple media resources for <video> or <audio> elements.
