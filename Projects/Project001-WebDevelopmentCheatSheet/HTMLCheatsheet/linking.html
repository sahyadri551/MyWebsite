HTML Linking

Definition

Linking connects web resources using the anchor <a> tag. Links can be internal (to the same site) or external (to another site).

Anchor Tag Attributes

  • href: Specifies the destination URL. This is the most important attribute.
  • target: Defines where to open the link.
    • _blank: Opens the link in a new tab or window.
    • _self: Opens the link in the same frame (default).
    • _parent: Opens the link in the parent frame.
    • _top: Opens the link in the full body of the window.
  • title: Provides extra information, shown as a tooltip on hover.
  • mailto:: Used in the href to create a link that opens the user's email client. use subject= and body= after ? to specify the email subject and body. Example: mailto:someone@example.com?subject=Hello&body=Message

Absolute vs. Relative Paths

Feature Absolute Path Relative Path
Format Full URL including protocol (e.g., https://) and domain. Path relative to the current file's location (e.g., about.html or ../images/pic.jpg).
Definition A full URL including protocol (https://) and domain. A partial path relative to the current file's location.
Use Case Linking to external websites. Linking to other pages within your own site.
Example https://www.google.com about.html or ../images/pic.jpg

Example Illustration

Here are different types of links in action.

<!-- External link to another website -->
<a href="https://www.google.com" target="_blank">Go to Google</a>

<!-- Internal link to another page on this site -->
<a href="index.html" title="Back to Home">Back to Home</a>

<!-- A mailto link to open an email client -->
<a href="mailto:someone@example.com">Contact Us</a>
  • External Link: Go to Google
  • Internal Link: Back to Home
  • Mailto Link: Contact Us
