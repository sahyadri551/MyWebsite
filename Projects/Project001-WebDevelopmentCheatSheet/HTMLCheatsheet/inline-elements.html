Inline Elements

Definition

An inline element does not start on a new line and only takes up as much width as necessary for its content. It is used to format parts of text within a block-level element.

Characteristics of Inline Elements:

  • Do not start on a new line.
  • Only take up as much width as their content requires.
  • Can be nested within block elements or other inline elements.
  • Do not disrupt the flow of text.
  • Commonly used for styling text, links, and small images.

List of Common Inline Elements

  • <span>: A generic inline container for styling.
  • <a>: An anchor tag for creating hyperlinks.
  • <b>, <strong>: For bolding text.
  • <i>, <em>: For italicizing or emphasizing text.
  • <img>: For embedding an image.
  • <code>: For marking up a piece of code.
