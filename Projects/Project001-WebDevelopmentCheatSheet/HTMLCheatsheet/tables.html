HTML Tables

Definition

HTML tables allow you to arrange data into rows and columns. They are used to display tabular data, such as financial reports, schedules, or product comparisons, in a structured and easy-to-read format.

Details & Usage

A table is created with the <table> tag. Inside it, you define table rows with <tr>, table headers with <th>, and table data cells with <td>. For better structure and accessibility, it's recommended to group header content in <thead>, body content in <tbody>, and footer content in <tfoot>.

List of All HTML Tags

  • <table>: Defines a table.
  • <thead>: Groups the header content in a table.
  • <tbody>: Groups the body content in a table.
  • <tfoot>: Groups the footer content in a table.
  • <tr>: Defines a row in a table.
  • <th>: Defines a header cell in a table. Text is bold and centered by default.
  • <td>: Defines a standard data cell in a table.
  • <caption>: Defines a table caption.
  • <colgroup>: Specifies a group of one or more columns for formatting.
  • <col>: Specifies column properties for each column within a <colgroup>.

List of All Possible HTML Tag Attributes

Table elements support Global Attributes. Key specific attributes include:

  • for <table>:
    • border: (Deprecated) Specifies the width of the border around the table. Use CSS instead.
  • for <th> and <td>:
    • colspan: Specifies the number of columns a cell should span.
    • rowspan: Specifies the number of rows a cell should span.
    • headers: Specifies one or more header cells a cell is related to for accessibility.
  • for <th>:
    • scope: Specifies whether a header cell is a header for a column, row, or group of columns/rows (col, row, colgroup, rowgroup).

Example Illustration

The code below shows a table with a header, body, and cells that span multiple columns and rows.

<table>
    <caption>Employee Schedule</caption>
    <thead>
        <tr>
            <th>Name</th>
            <th>Monday</th>
            <th>Tuesday</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Alice</td>
            <td>9 AM - 5 PM</td>
            <td>Off</td>
        </tr>
        <tr>
            <td>Bob</td>
            <td colspan="2">On-site Training</td>
        </tr>
    </tbody>
</table>
