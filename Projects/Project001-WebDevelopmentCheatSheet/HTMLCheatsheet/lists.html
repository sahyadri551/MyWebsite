HTML Lists

Definition

HTML lists allow web developers to group a set of related items in lists. There are three types of lists in HTML: unordered lists, ordered lists, and description lists.

Details & Usage

Lists are fundamental for organizing content, making it more readable and structured. Unordered lists (<ul>) are for items where the order does not matter (like a shopping list). Ordered lists (<ol>) are for items where the order is important (like step-by-step instructions). Description lists (<dl>) are used to display name-value pairs, such as terms and their definitions.

List of All Possible HTML Tags

  • <ul>: Defines an unordered list (bullet points).
  • <ol>: Defines an ordered list (numbered).
  • <li>: Defines a list item within a <ul> or <ol>.
  • <dl>: Defines a description list.
  • <dt>: Defines a term (a name) in a description list.
  • <dd>: Describes each term in a description list.

List of All Possible HTML Tag Attributes

Most list-related tags primarily use Global Attributes. However, some have unique attributes:

  • for <ol>:
    • reversed: A boolean attribute that specifies the list order should be descending.
    • start: A number that specifies the starting value of the list.
    • type: Specifies the kind of marker to use ('1', 'A', 'a', 'I', 'i').
  • for <li>:
    • value: For <ol> lists, this attribute specifies a new number for the current list item.
  • for <ul>:
    • type: (Deprecated) Specifies the bullet style ('disc', 'circle', 'square'). CSS is now preferred.

Example Illustration

The code below shows the three different types of lists.

<h3>Unordered List (Groceries)</h3>
<ul>
    <li>Milk</li>
    <li>Bread</li>
    <li>Cheese</li>
</ul>

<h3>Ordered List (Instructions)</h3>
<ol start="1" type="1">
    <li>Gather ingredients</li>
    <li>Mix them together</li>
    <li>Bake for 30 minutes</li>
</ol>

<h3>Description List (Web Terms)</h3>
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>
