JavaScript Math Object

Introduction to Math

The JavaScript `Math` object is a built-in object that provides properties and methods for mathematical constants and functions. Unlike other global objects, `Math` is not a constructor. All properties and methods of `Math` are static and can be called by using `Math` as an object without creating it.

Mathematical Constants

The `Math` object provides several built-in mathematical constants as properties.

console.log(Math.PI);      // 3.141592653589793 (Ratio of a circle's circumference to its diameter)
console.log(Math.E);         // 2.718281828459045 (Euler's number)
console.log(Math.SQRT2);   // 1.4142135623730951 (Square root of 2)
console.log(Math.LN10);      // 2.302585092994046 (Natural logarithm of 10)

Rounding Methods

  • Math.round(x): Returns the value of `x` rounded to the nearest integer.
  • Math.ceil(x): Returns the smallest integer greater than or equal to `x` (rounds up).
  • Math.floor(x): Returns the largest integer less than or equal to `x` (rounds down).
  • Math.trunc(x): Returns the integer part of `x` by removing any fractional digits.
console.log(Math.round(4.7)); // 5
console.log(Math.round(4.4)); // 4
console.log(Math.ceil(4.2));  // 5
console.log(Math.floor(4.8)); // 4
console.log(Math.trunc(4.9)); // 4

Power and Root Methods

  • Math.pow(x, y): Returns the value of `x` to the power of `y`.
  • Math.sqrt(x): Returns the square root of `x`.
  • Math.cbrt(x): Returns the cube root of `x`.
  • Math.exp(x): Returns $e^x$, where $e$ is Euler's number.
console.log(Math.pow(8, 2));   // 64
console.log(Math.sqrt(64));  // 8
console.log(Math.cbrt(125)); // 5

Min, Max, and Absolute Value

  • Math.min(...): Returns the lowest-valued number passed into it.
  • Math.max(...): Returns the largest-valued number passed into it.
  • Math.abs(x): Returns the absolute (positive) value of `x`.
console.log(Math.min(0, 150, 30, 20, -8, -200)); // -200
console.log(Math.max(0, 150, 30, 20, -8, -200)); // 150
console.log(Math.abs(-7.8));                 // 7.8

Random Number Generation

Math.random() returns a floating-point, pseudo-random number in the range from 0 (inclusive) up to but not including 1.

You can combine it with other methods to generate random numbers in a specific range.

// Returns a random number between 0 and 1 (e.g., 0.12345)
console.log(Math.random());

// Returns a random integer from 0 to 9
console.log(Math.floor(Math.random() * 10));

// Returns a random integer from 1 to 10
console.log(Math.floor(Math.random() * 10) + 1);

Trigonometric and Logarithmic Methods

The `Math` object provides a full suite of trigonometric and logarithmic functions.

  • Trigonometric methods like Math.sin(), Math.cos(), and Math.tan() take angles in **radians**.
  • Logarithmic methods include Math.log() (natural log), Math.log10(), and Math.log2().
// Calculate sine of 90 degrees
let angleInDegrees = 90;
let angleInRadians = angleInDegrees * Math.PI / 180;
console.log(Math.sin(angleInRadians)); // 1

// Logarithmic functions
console.log(Math.log(Math.E));  // 1 (Natural log)
console.log(Math.log10(100)); // 2 (Base-10 log)
