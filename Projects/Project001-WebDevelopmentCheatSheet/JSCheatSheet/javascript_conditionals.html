JavaScript Conditional Statements

Definition

Conditional statements are used to perform different actions based on different conditions. They allow your code to make decisions and execute specific blocks of code only when a certain condition is met. This control over the flow of execution is fundamental to programming.

Detailed Explanation of if, if else, else if

The if Statement

The if statement executes a block of code if a specified condition is true.

Syntax:

if (condition) {
  // block of code to be executed if the condition is true
}

Example:

let hour = 14;

if (hour < 18) {
  console.log("Good day");
}
// Output: Good day

The if...else Statement

The if...else statement executes one block of code if the condition is true and another block if the condition is false.

Syntax:

if (condition) {
  // block of code to be executed if the condition is true
} else {
  // block of code to be executed if the condition is false
}

Example:

let temperature = 30;

if (temperature > 25) {
  console.log("It's a hot day!");
} else {
  console.log("It's a pleasant day.");
}
// Output: It's a hot day!

The else if Statement

The else if statement specifies a new condition to test if the first condition is false. You can have multiple else if blocks.

Syntax:

if (condition1) {
  // block of code for condition1
} else if (condition2) {
  // block of code for condition2
} else {
  // block of code if no conditions are met
}

Example:

let score = 85;

if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");
} else if (score >= 70) {
  console.log("Grade: C");
} else {
  console.log("Grade: F");
}
// Output: Grade: B

Nested if...else Statement

You can use an if or else if statement inside another if or else if statement. This is known as nesting.

Example:

This program checks if a number is positive, negative, or zero, and if it's positive, it also checks if it's even or odd.

let num = 10;

if (num >= 0) {
  if (num === 0) {
    console.log("The number is zero.");
  } else {
    console.log("The number is positive.");
    if (num % 2 === 0) {
      console.log("It is also an even number.");
    } else {
      console.log("It is also an odd number.");
    }
  }
} else {
  console.log("The number is negative.");
}
// Output:
// The number is positive.
// It is also an even number.

The switch Statement

The switch statement is used to perform different actions based on different conditions. It evaluates an expression, matching the expression's value against a series of case clauses, and executes statements after the first clause with a matching value, until a break statement is encountered.

  • case: Defines a block of code to be executed if the expression matches the case's value.
  • break: Stops the execution inside the switch block. If omitted, execution continues into the next case.
  • default: Specifies the code to run if there is no case match.

Example:

let day = new Date().getDay(); // Returns a number (0=Sun, 1=Mon, ...)
let dayName;

switch (day) {
  case 0:
    dayName = "Sunday";
    break;
  case 1:
    dayName = "Monday";
    break;
  case 2:
    dayName = "Tuesday";
    break;
  case 3:
    dayName = "Wednesday";
    break;
  case 4:
    dayName = "Thursday";
    break;
  case 5:
    dayName = "Friday";
    break;
  case 6:
    dayName = "Saturday";
    break;
  default:
    dayName = "Unknown Day";
}

console.log("Today is " + dayName);

Nested switch Statement

A switch statement can be nested inside another switch statement. This can be useful for evaluating more complex, multi-level conditions.

Example: Even-Odd Number Check Program

This program checks if a number is between 1 and 10, and then uses a nested switch to determine if the number is even or odd.

let num = 7;
let message = "";

// Outer switch checks if the number is within a certain range (e.g., a single digit)
switch (num > 0 && num < 10) {
    case true:
        message += "The number is a single positive digit. ";
        
        // Inner switch checks if the number is even or odd
        switch (num % 2) {
            case 0:
                message += "It is an EVEN number.";
                break;
            case 1:
                message += "It is an ODD number.";
                break;
        }
        break;

    case false:
        message = "The number is not a single positive digit.";
        break;
        
    default:
        message = "Invalid input.";
        break;
}

console.log(message);
// Output: The number is a single positive digit. It is an ODD number.
