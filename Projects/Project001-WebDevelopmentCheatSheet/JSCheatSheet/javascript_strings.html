JavaScript Strings

Introduction to Strings

A string in JavaScript is a sequence of characters used to represent text. Strings are primitive data types and are **immutable**, which means that once a string is created, it cannot be changed. Methods that appear to modify a string actually return a new string with the changes.

let greeting = "Hello, World!"; // Using double quotes
let name = 'Alice';         // Using single quotes
let message = `Hello, ${name}!`; // Using template literals (ES6)

Properties and Character Access

  • .length: A property that returns the number of characters in the string.
  • .charAt(index): Returns the character at a specified index.
  • Bracket Notation [index]: A modern way to access the character at a specified index.
const str = "The Sample String";
console.log(str.length); // 17
console.log(str.charAt(4)); // 'S'
console.log(str[4]);         // 'S'

Changing Case

  • .toUpperCase(): Returns a new string with all characters converted to uppercase.
  • .toLowerCase(): Returns a new string with all characters converted to lowercase.
const str = "The Sample String";
console.log(str.toUpperCase()); // "THE SAMPLE STRING"
console.log(str.toLowerCase()); // "the sample string"

Extracting Substrings

  • .slice(startIndex, endIndex): Extracts a section of a string and returns it as a new string. The `endIndex` is not included. Can accept negative indices.
  • .substring(startIndex, endIndex): Similar to `slice`, but cannot accept negative indices.
  • .substr(startIndex, length): (Deprecated) Returns a specified number of characters from a `startIndex`.
const str = "The Sample String";
console.log(str.slice(4, 10)); // "Sample"
console.log(str.slice(-6));   // "String"
console.log(str.substring(4, 10));// "Sample"

Searching and Finding

  • .indexOf(searchValue): Returns the index of the first occurrence of a value, or -1 if not found.
  • .lastIndexOf(searchValue): Returns the index of the last occurrence of a value, or -1 if not found.
  • .includes(searchValue): Returns `true` if the string contains the specified value, otherwise `false`.
  • .startsWith(searchValue): Returns `true` if the string begins with the specified value.
  • .endsWith(searchValue): Returns `true` if the string ends with the specified value.
  • .search(regexp): Searches for a match against a regular expression and returns the index of the match, or -1.
const str = "The Sample String";
console.log(str.indexOf('S'));       // 4
console.log(str.includes('Sample')); // true
console.log(str.startsWith('The'));  // true

Modifying and Transforming

  • .replace(searchValue, newValue): Returns a new string with the first match of `searchValue` replaced by `newValue`. Can use a regular expression for more powerful replacements.
  • .replaceAll(searchValue, newValue): Returns a new string with all matches replaced.
  • .concat(str1, str2, ...): Joins two or more strings and returns a new combined string. The `+` operator is often more convenient.
  • .trim(): Removes whitespace from both ends of a string.
  • .repeat(count): Returns a new string containing the specified number of copies of the original string.
  • .padStart(targetLength, padString) / .padEnd(...): Pads the current string from the start/end with another string until it reaches the target length.
const str = "The Sample String";
console.log(str.replace('Sample', 'Example')); // "The Example String"

const greeting = "   Hello World!   ";
console.log(greeting.trim()); // "Hello World!"

console.log('5'.padStart(3, '0')); // "005"

Converting to an Array

  • .split(separator): Divides a string into an ordered list of substrings by searching for a `separator`, puts these substrings into an array, and returns the array.
const str = "The Sample String";
const words = str.split(' ');
console.log(words); // ["The", "Sample", "String"]

const csv = 'apple,banana,cherry';
const fruits = csv.split(',');
console.log(fruits); // ["apple", "banana", "cherry"]

Regular Expression Methods

  • .match(regexp): Retrieves the result of matching a string against a regular expression.
  • .matchAll(regexp): Returns an iterator of all results matching a string against a regular expression, including capturing groups.
const str = "The rain in SPAIN stays mainly in the plain.";
const matches = str.matchAll(/ain/gi);

for (const match of matches) {
  console.log(`Found '${match[0]}' at index ${match.index}`);
}
// Found 'ain' at index 5
// Found 'AIN' at index 14
// Found 'ain' at index 25
// Found 'ain' at index 43

Deprecated HTML Wrapper Methods

JavaScript used to have methods that would wrap a string in HTML tags (e.g., .bold(), .link(), .fontcolor()). These methods are now deprecated and should not be used in modern web development. Instead, you should use the DOM API (e.g., document.createElement) to create and style HTML elements.
