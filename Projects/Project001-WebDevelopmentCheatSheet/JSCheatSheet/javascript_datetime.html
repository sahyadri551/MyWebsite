JavaScript Date & Time

Introduction to the Date Object

The JavaScript `Date` object is used to work with dates and times. Date objects are created with the `new Date()` constructor. A `Date` object represents a single moment in time, stored as the number of milliseconds since the Unix Epoch (January 1, 1970, 00:00:00 UTC).

Creating Date Objects

There are four ways to create a `Date` object:

  1. new Date(): Creates a date object with the current date and time.
  2. new Date(milliseconds): Creates a date object from the number of milliseconds since the epoch.
  3. new Date(dateString): Creates a date object from a date string (e.g., "2025-08-26").
  4. new Date(year, month, day, hours, minutes, seconds, ms): Creates a date object with a specified date and time. Note: The month is 0-indexed (0 = January, 11 = December).
const now = new Date();
console.log(now); // Shows current date and time

const specificDate = new Date(2025, 7, 26); // August 26, 2025 (month is 0-indexed)
console.log(specificDate);

Getting Date and Time Components

Once you have a `Date` object, you can retrieve its components using various "get" methods.

  • .getFullYear(): Gets the four-digit year (e.g., 2025).
  • .getMonth(): Gets the month (0-11).
  • .getDate(): Gets the day of the month (1-31).
  • .getDay(): Gets the day of the week (0=Sunday, 6=Saturday).
  • .getHours(), .getMinutes(), .getSeconds(): Gets time components.
  • .getTime(): Gets the number of milliseconds since the epoch.
  • Date.now(): A static method that returns the current timestamp in milliseconds.
const d = new Date('2025-08-26T10:30:00');
console.log(`Year: ${d.getFullYear()}`);      // Year: 2025
console.log(`Month: ${d.getMonth()}`);        // Month: 7 (August)
console.log(`Date: ${d.getDate()}`);         // Date: 26
console.log(`Minutes: ${d.getMinutes()}`); // Minutes: 30

Setting Date and Time Components

Similarly, you can modify a date object using various "set" methods. These methods modify the `Date` object in place.

  • .setFullYear(year)
  • .setMonth(month)
  • .setDate(day)
  • .setHours(hour), etc.
const eventDate = new Date();
eventDate.setFullYear(2026);
eventDate.setMonth(11); // December
eventDate.setDate(25);
console.log(eventDate.toDateString()); // "Thu Dec 25 2026" (day of week may vary)

Formatting Dates to Strings

Date objects can be converted into human-readable strings in several standard formats.

  • .toString(): Full standard string representation.
  • .toDateString(): Date portion only (e.g., "Tue Aug 26 2025").
  • .toTimeString(): Time portion only.
  • .toLocaleString(): A locale-sensitive representation of the date and time.
  • .toLocaleDateString(): Locale-sensitive date portion (e.g., "8/26/2025" in US).
  • .toISOString(): A standardized format (YYYY-MM-DDTHH:mm:ss.sssZ) often used in APIs.
const d = new Date();
console.log(d.toDateString());
console.log(d.toLocaleDateString('en-IN')); // Formatted for India (e.g., 26/8/2025)
console.log(d.toISOString());
