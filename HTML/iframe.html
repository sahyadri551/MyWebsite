Attributes of the iframe element:

  • src - Specifies the URL of the document to embed
  • width - Specifies the width of the iframe
  • height - Specifies the height of the iframe
  • frameborder - Specifies whether or not to display a border around the iframe (deprecated)
  • allowfullscreen - Specifies that the iframe can be put into full-screen mode
  • loading - Specifies whether the iframe should be loaded immediately or lazily (e.g., "lazy")
  • sandbox - Applies extra restrictions to the content in the iframe, such as preventing scripts from running or forms from submitting
  • title - Provides a title for the iframe content, which is important for accessibility
  • allow - Specifies features that the iframe can use, such as "camera", "microphone", or "geolocation"
  • referrerpolicy - Specifies the referrer information to be sent when navigating the iframe
  • srcdoc - Specifies the HTML content of the iframe directly, instead of a URL

    • allow attributes:

    • camera - Allows access to the camera
    • microphone - Allows access to the microphone
    • geolocation - Allows access to the geolocation
    • payment - Allows access to payment methods
    • fullscreen - Allows the iframe to be put into full-screen mode
    • clipboard-write - Allows writing to the clipboard
    • encrypted-media - Allows playback of encrypted media
    • picture-in-picture - Allows the video to be played in picture-in-picture mode
    • autoplay - Allows autoplay of media
    • xr-spatial-tracking - Allows access to XR spatial tracking
    • accelerometer - Allows access to the accelerometer
    • gyroscope - Allows access to the gyroscope
    • magnetometer - Allows access to the magnetometer
    • usb - Allows access to USB devices
    • vibrate - Allows access to vibration features

Open Links in iframe

  • To open links in an iframe, you can set the target attribute of the link to the name of the iframe.
  • Example: Open Example in iframe
  • Make sure the iframe has a name attribute that matches the target.
